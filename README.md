# Welcome to Seek Friend Apps Dev environnement

## Install git tuto
#### create new repository on github.com
- git init
- git add *
- git commit -m 'first commit'
- git remote add origin @git
- git push -u origin master

- git commit -m 'v1.0'
- git push

#### clone repository
- git clone @git
- git pull
- git commit -m 'v1.1'
- git push
